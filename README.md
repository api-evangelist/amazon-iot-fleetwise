# Amazon IoT FleetWise (amazon-iot-fleetwise)
AWS IoT FleetWise is a fully managed service that makes it easier to collect, transform, and transfer vehicle data to the cloud in near-real time. It provides tools for modeling vehicle signals and configuring intelligent data collection from vehicle fleets at scale.

## APIs

### AWS IoT FleetWise API
The AWS IoT FleetWise API provides programmatic access to create and manage vehicle models, fleets, signal catalogs, campaigns, and data collection rules for vehicle fleet data ingestion.
